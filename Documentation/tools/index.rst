=====
Tools
=====

This section provides documentation for the various tools and utilities
associated with the **Dawn** framework.

The Python tooling is distributed as a small core package (``dawnpy``)
plus a family of extension packages. Each page below documents one tool:
purpose, install, command surface, and tests.
